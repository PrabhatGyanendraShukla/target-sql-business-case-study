# 🛒 Target Brazil SQL Business Case Study

## Project Overview

This project analyzes Target Brazil's e-commerce operations using SQL and Google BigQuery.

The analysis is based on 100,000+ customer orders placed between 2016 and 2018 across multiple datasets.

## Business Objective

The objective is to identify business insights related to:

- Customer purchasing behavior
- Sales trends
- Payment methods
- Freight costs
- Delivery performance
- State-wise business performance

## Dataset

- 100,000+ Orders
- 8 Relational Tables
- Google BigQuery
- SQL

## Skills Demonstrated

- SQL
- JOINs
- CTEs
- Window Functions
- Aggregate Functions
- Date Functions
- Business Analysis
